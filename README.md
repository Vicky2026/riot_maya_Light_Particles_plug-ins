riot _ Maya _ Light _ Particles _ Import _ Export _ scripts
用于导入英雄联盟S4.nvr地图模型配置文件，旧版本的Light.dat灯光文件，和Particles.dat，粒子数据位置文件数据,并导出到新版本英雄联盟.py .bin粒子结构矩形正位置的maya插件。

要使用这个插件只需要把解压的一整个文件夹复制到文档📄目录

![狙击手2025年6月18日17月50日27日](https://github.com/user-attachments/assets/25bcbd0b-2eab-48a3-a2fb-d127ddfad032)

在maya窗口插件管理器中勾选激活，脚本名称以及工具栏和工具架按钮将会出现。
![狙击手2025年6月18日18](https://github.com/user-attachments/assets/45a678f0-ff0a-4ba7-a1c0-f838ea503b85)

按钮1导入Light.dat灯光数据
![1](https://github.com/user-attachments/assets/a7b6e5c9-39e9-4a2f-9fe7-2808ddd678da)

您需要设置渲染器中灯光数量的限制。需要注意的是,当你在灯光设置中使用选定的灯光时,不要选择整个组。如果你的电脑配置不够,你的maya会卡死。

按钮2导入Particles.dat.粒子位置数据
![4](https://github.com/user-attachments/assets/ab0638ac-5ca6-441c-ac81-806e4ba4fc09)
